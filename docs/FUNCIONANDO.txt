================================================================================
                    PRÓXIMOS PASSOS
================================================================================

1. CRIAR USUÁRIOS NO AUTH0
---------------------------
- Ir em: Auth0 → User Management → Users
- Criar usuário "barista"
- Criar usuário "manager"
- OU usar usuário atual (Google OAuth)

2. CONFIGURAR ROLES
-------------------
- Auth0 → User Management → Roles
- Criar Role: "Barista" 
- Criar Role: "Manager"

3. ATRIBUIR PERMISSÕES AOS ROLES
--------------------------------
Barista:
  - get:drinks
  - get:drinks-detail

Manager:
  - get:drinks
  - get:drinks-detail
  - post:drinks
  - patch:drinks
  - delete:drinks

4. ATRIBUIR ROLES A USUÁRIOS
-----------------------------
- Users → selecionar usuário → Roles
- Atribuir role apropriada

5. TESTAR COM JWT COM PERMISSÕES
---------------------------------
- Fazer logout
- Fazer login novamente
- JWT deve ter "permissions": ["get:drinks", etc]
