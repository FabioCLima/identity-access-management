## 🚀 Como Rodar Agora

### Terminal 1: Backend

```bash
cd backend
./start_server.sh
```

### Terminal 2: Frontend

```bash
cd frontend
export NODE_OPTIONS=--openssl-legacy-provider
ionic serve
```

### Navegador

Abrir: **http://localhost:8100**

**O que deve acontecer:**
1. Página abre
2. Clicar em "Login"
3. Fazer login na Auth0
4. Ver drinks aparecem
5. Permissions funcionam (botões corretos)
